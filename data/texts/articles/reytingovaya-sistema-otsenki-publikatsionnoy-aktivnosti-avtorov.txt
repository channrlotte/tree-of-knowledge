Аннотация
В работе рассматривается построение рейтинга публикационной активности авторов на основе показателей Российского Индекса Научного Цитирования (РИНЦ). Для определения рейтинга используется метод линейной трансформации, а также предполагается ввод весовых показателей, для того чтобы каждый эксперт мог установить насколько значимым является каждый из показателей публикационной активности авторов. Реализована система определения рейтинга среди профессорско-преподавательского состава (ППС) кафедры АСОИУ Казанского национального исследовательского технического университета им. А.Н. Туполева-КАИ (КНИТУ-КАИ) по публикационной активности с использованием основных показателей. This work shows the problem of determining the rating of the authors' publication activity on the basis of the Russian Scientific Citation Index (RSCI). To determine the rating, the method of linear transformation is used, and it is also proposed to enter weight indicators, so that each expert can establish how significant each of the indicators of the authors' publication activity is. A system for determining the rating among the teaching staff of the cathedra ASOIU of Kazan National Research Technical University was established. A.N. Введение
Качество образования существенно влияет на развитие науки. Для достижения высокого качества образования, необходимо, чтобы сотрудники высших учебных заведений активно участвовали в научных исследованиях. Для того чтобы оценить публикации по научным исследованиям система РИНЦ ввела целый ряд показателей для оценки авторов. [1-3] Но каждый показатель в отдельности недостаточно
информативен и не дает исчерпывающую оценку публикационной активности авторов. Поэтому возникает необходимость в разработке такой системы определения рейтинга, которая смогла бы на основе выбранных показателей системы РИНЦ выстраивать рейтинг авторов согласно их достижениям в сфере научных публикаций.
Для того, чтобы спроектировать и реализовать рейтинговую систему публикационной активности авторов, необходимо последовательно выполнить каждый из следующих шагов:
1)	Анализ всех имеющихся показателей авторов в системе РИНЦ и выявление среди них ключевых показателей для оценки публикационной активности авторов;
2)	Нормирование выбранных ключевых показателей для получения безразмерных величин для сведения к единой системе сравнения;
3)	Введение возможности выставления независимой оценки экспертом для каждого из ключевых показателей;
4)	Вычисление итогового суммарного значения рейтинга из полученных на предыдущих шагах взвешенных нормированных показателей для каждого автора, сортировка авторов по убыванию этих значений.

Нормирование показателей для получения безразмерных величин показателей.
Под нормированием выбранных на предыдущем этапе показателей публикационной активности авторов понимается приведение каждого из показателей к единому безразмерному виду. В качестве метода нормирования в разрабатываемой системе рейтинга авторов применяется один из наиболее употребляемых и понятных способов приведения критериев к безразмерному виду - метод линейной трансформации.

Таким образом, каждый показатель после линейной трансформации избавляется от своих единиц измерений и становится безразмерной величиной, а значит теперь между показателями можно проводить различные арифметические операции для получения единого значения рейтинга.
Выставление независимой оценки экспертом каждому показателю.
Для того, чтобы увеличить гибкость разрабатываемой рейтинговой системы оценки публикационной активности, в систему введена возможность выставления независимой оценки экспертом каждому из показателей.
Экспертный метод оценки показателей основывается на использовании опыта и интуитивных знаниях эксперта-специалиста. Экспертный метод используется в тех случаях, когда применение таких методов объективного определения значений, как инструментальная, эмпирическая или расчетная оценка, является затруднительным.
При экспертном методе оценки показатели определяют в безразмерных величинах. В случае, если результат оценки (экспертного измерения) эксперты представляют в виде ранжированного ряда, то численное определение оценок экспертов состоит в следующем [7]:
•	Все объекты оценки (показатели) произвольно нумеруются.
•	Эксперт вводит в систему свою субъективную оценку каждому параметру
•	Рейтинг авторов упорядочиваются по убыванию.
Анализируя полученные экспертным методом оценки показателей, можно не только указать, какой показатель весомее и значительнее, чем другие, но и во сколько.
Вычисление значений рейтинга и сортировка авторов.
Расчет итоговых значений рейтинга происходит по следующему алгоритму. На первом шаге все показатели нормируются методом линейной трансформации, на втором шаге полученные нормированные показатели умножаются на соответствующий вес, введенный пользователем. Затем полученные произведения суммируются, и результат этой суммы является итоговым значением рейтинга. Чем больше итоговое значение, тем выше занимает место автор в рейтинге публикационной активности. Единственным исключением является показатель числа самоцитирований. Он может или вычитаться из суммарного значения рейтинга, или не влиять на него в зависимости от своего значения. Если параметр числа самоцитирований меньше или равен 40%, то он не учитывается в суммировании показателей для определения рейтинга, а если больше 40%, то он нормируется, умножается на вес экспертной оценки и вычитается из суммарного значения рейтинга. Такое различие обусловлено тем, что этот показатель имеет две стороны. С одной стороны, его значение указывает на то, что автор продолжительное время работает в одной предметной области и ссылается на предыдущие статьи за разъяснением, с другой, этот параметр может свидетельствовать о том, что автор пытается увеличить свой рейтинг за счёт фиктивного цитирования собственных работ. Чтобы исключить возможность искусственного увеличения рейтинга автора, предложена формула расчета. Итоговая формула для расчета рейтинга выглядит следующим образом:
Хш - Хмк — нормированные значений ключевых показателей публикационной активности автора,
- Ьм — веса экспертной оценки для каждого показателя, Х5Ы — нормированный показатель числа самоцитирований, Х5 — показатель числа самоцитирований,
Ъ8 — вес экспертной оценки для показателя числа самоцитирований, М — количество ключевых показателей публикационной активности авторов, N — тип нормированности значений, Я — рейтинг автора.

Заключение
В статье изложен алгоритм построения рейтинга авторов на примере профессорско-преподавательского состава кафедры АСОИУ КНИТУ-КАИ по публикационной активности с использованием основных показателей РИНЦ в рейтинговой системе публикационной активности. Сама система может быть применена к любой базе данных авторов. Предложенный алгоритм позволяет оценить авторов по ключевым показателям РИНЦ, тем самым позволяя в достаточной мере показать активность авторов.
Для расчета оценки публикационной активности авторов предложенным методом применяется нормирование показателей публикационной активности методом линейной трансформации и присваивание каждому показателю веса экспертной оценки. В настоящее время область публикационной научной активности исследована недостаточно полно для расчета оптимальных параметров оценки авторов и построения рейтинга лучших авторов. Поэтому следует отметить, что результат оценки авторов в разработанной рейтинговой системе базируется на выбранных показателях, суммирование которых вычисляет значения для построения рейтинга авторов.
